Quick Setup Guide

1. Install Atlas bot on your VPS
2. Clone DashView repo to /opt/dashview
3. Set Bot Directory to your Atlas install path
4. Set Bot Name to whatever you called your bot
5. Set up DuckDNS + SSL (see HTTPS_SETUP.md)
6. Save & Restart
